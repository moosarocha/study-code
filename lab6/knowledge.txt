เครื่องคอมพิวเตอร์เปิดไม่ติด,พัดลม Power Supply ไม่หมุน => สงสัยว่า Power Supply ชำรุด
สงสัยว่า Power Supply ชำรุด => แนะนำให้เปลี่ยน Power Supply

เครื่องคอมพิวเตอร์เปิดติด,ไม่มีภาพบนหน้าจอ,ได้ยินเสียง Beep 1 ยาว 2 สั้น => สงสัยว่าการ์ดแสดงผลมีปัญหา
สงสัยว่าการ์ดแสดงผลมีปัญหา => แนะนำให้ตรวจสอบหรือเปลี่ยนการ์ดแสดงผล

เครื่องคอมพิวเตอร์เปิดติด,ไม่มีภาพบนหน้าจอ,ไม่มีเสียง Beep => สงสัยว่า RAM มีปัญหา
สงสัยว่า RAM มีปัญหา => แนะนำให้ถอดและเสียบ RAM ใหม่
ถอดและเสียบ RAM ใหม่แล้ว,ยังไม่มีภาพบนหน้าจอ => แนะนำให้เปลี่ยน RAM

เครื่องรีสตาร์ตเองเป็นระยะ,อุณหภูมิ CPU สูงกว่า 90 องศาเซลเซียส => สงสัยว่าระบบระบายความร้อนของ CPU มีปัญหา
สงสัยว่าระบบระบายความร้อนของ CPU มีปัญหา => แนะนำให้ทำความสะอาดพัดลมและเปลี่ยนซิลิโคนระบายความร้อน

ฮาร์ดดิสก์มีเสียงดังผิดปกติ,เครื่องทำงานช้าผิดปกติ => สงสัยว่าฮาร์ดดิสก์ใกล้เสีย
สงสัยว่าฮาร์ดดิสก์ใกล้เสีย => แนะนำให้สำรองข้อมูลทันที
สำรองข้อมูลเรียบร้อยแล้ว => แนะนำให้เปลี่ยนฮาร์ดดิสก์

เชื่อมต่อ Wi-Fi ไม่ได้,Network Adapter มีเครื่องหมายตกใจ => สงสัยว่าไดรเวอร์การ์ดเครือข่ายมีปัญหา
สงสัยว่าไดรเวอร์การ์ดเครือข่ายมีปัญหา => แนะนำให้ติดตั้งไดรเวอร์ใหม่

เครื่องเปิดติด,BIOS แจ้ง CPU Fan Error => สงสัยว่าพัดลม CPU ไม่ทำงาน
สงสัยว่าพัดลม CPU ไม่ทำงาน => แนะนำให้ตรวจสอบหรือเปลี่ยนพัดลม CPU

เครื่องเปิดติด,USB ทุกช่องใช้งานไม่ได้ => สงสัยว่าเมนบอร์ดมีปัญหา
สงสัยว่าเมนบอร์ดมีปัญหา => แนะนำให้ตรวจสอบหรือเปลี่ยนเมนบอร์ด

เครื่องเปิดติด,มีเสียง Beep ต่อเนื่อง => สงสัยว่า RAM เสีย
สงสัยว่า RAM เสีย => แนะนำให้เปลี่ยน RAM

เครื่องเปิดติด,เข้าสู่ระบบปฏิบัติการไม่ได้,พบข้อความ Operating System not found => สงสัยว่าอุปกรณ์บูตมีปัญหา
สงสัยว่าอุปกรณ์บูตมีปัญหา => แนะนำให้ตรวจสอบลำดับการบูตและฮาร์ดดิสก์

เครื่องเปิดติด,วันที่และเวลาไม่ถูกต้องทุกครั้ง => สงสัยว่าแบตเตอรี่ CMOS หมด
สงสัยว่าแบตเตอรี่ CMOS หมด => แนะนำให้เปลี่ยนแบตเตอรี่ CMOS

เครื่องเปิดติด,จอฟ้า (Blue Screen) บ่อย => สงสัยว่าไดรเวอร์หรือ RAM มีปัญหา
สงสัยว่าไดรเวอร์หรือ RAM มีปัญหา => แนะนำให้ตรวจสอบไดรเวอร์และทดสอบ RAM
